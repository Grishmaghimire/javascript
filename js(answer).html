JavaScript & jQuery Q&A

An event in JavaScript is an action or occurrence that happens in the web browser, which can be detected and responded to using JavaScript.

Examples:

  • Clicking a button → onclick
  • Pressing a key → onkeydown
  • Hovering over an element → onmouseover

There are three main ways to handle events:

MethodDescriptionExample
Inline event handling Define the event inside HTML. <button onclick="alert('Clicked!')">Click Me</button>
DOM property method Assign function to element property. document.getElementById('btn').onclick = function(){ alert('Clicked!'); };
addEventListener() Attach multiple handlers without overwriting. document.getElementById('btn').addEventListener('click', function(){ alert('Clicked!'); });

Keyboard events allow JavaScript to respond when a user interacts with the keyboard.

EventDescriptionExample
keydownTriggered when key is presseddocument.addEventListener('keydown', e => console.log(e.key));
keyupTriggered when key is releaseddocument.addEventListener('keyup', e => console.log(e.key));
keypressTriggered when key is pressed and held (deprecated)document.addEventListener('keypress', e => console.log(e.key));

Mouse events allow interaction using the mouse.

EventDescriptionExample
clickFires when element is clickedelement.addEventListener('click', myFunction);
dblclickFires on double-clickelement.addEventListener('dblclick', myFunction);
mouseoverFires when mouse enters elementelement.addEventListener('mouseover', myFunction);
mouseoutFires when mouse leaves elementelement.addEventListener('mouseout', myFunction);

jQuery is a fast, small, and feature-rich JavaScript library that simplifies DOM manipulation, event handling, animation, and Ajax. It is used because it requires less code and handles cross-browser compatibility automatically.

You can include jQuery using a CDN:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>

Common jQuery event handling methods:

  • .click(): Trigger function on click
  • .submit(): Trigger function on form submit
  • .hover(): Trigger function on mouse enter/leave
  • .append(): Add content at end
  • .prepend(): Add content at beginning
  • .before(): Add content before element
  • .after(): Add content after element
MethodPurposeExample
.html()Get or set HTML content$('#div').html('<b>Bold</b>');
.text()Get or set plain text$('#div').text('Hello World');
.css()Get or set CSS properties$('#div').css('color','red');
.val()Get or set form element values$('#input').val('Hello');

Use jQuery methods to control visibility:

  • .show(): Display a hidden element
  • .hide(): Hide an element
  • .toggle(): Toggle visibility

Example of using JavaScript to handle form input and display output:

document.getElementById('myForm').addEventListener('submit', function(e){
    e.preventDefault();
    let name = document.getElementById('name').value;
    document.getElementById('greeting').innerText = 'Hello, ' + name + '!';
});
